# 🖧 My Cisco Packet Tracer Lab Repository

Welcome to my Cisco networking lab repository! This is my personal collection of Packet Tracer labs covering fundamental to advanced networking concepts.

---

## 📁 Labs Included

| # | File | Description | Protocols/Technologies | Devices |
|---|------|-------------|------------------------|---------|
| 1 | ICMP.ARP.pkt | ARP and ICMP connectivity test | ARP, ICMP | 2x PCs, 2x 2960 switches |
| 2 | static-routing-lab.pkt | Static routing between 3 networks | ICMP, Static Routing | Router0, 4x PCs, 2960 switch |
| 3 | vlan-segmentation-lab.pkt | VLAN segmentation with multiple PCs | VLAN, Ethernet | 50+ PCs, managed switches |
| 4 | stp-bpduguard-lab.pkt | STP, Root Bridge election, BPDU Guard | STP, BPDU Guard, 802.1D | 3x switches |
| 5 | rstp-root-bridge-lab.pkt | RSTP root bridge election, port priority | RSTP (802.1w) | Multiple switches |
| 6 | layer3-intervlan-routing-lab.pkt | Layer 3 switch inter-VLAN routing with SVI, /26 subnetting | VLAN, Trunk (802.1Q), SVI, Static Routing | 2 switches (L2+L3), 1 router, 8 PCs |

---

## 🎯 Lab Highlights

### Lab 6: Layer 3 Inter-VLAN Routing
- **VLANs:** 10, 20, 30
- **Subnetting:** /26 (62 usable hosts per VLAN)
- **Routing:** SVI on Layer 3 switch + static routing
- **Trunk:** 802.1Q between switches

---

## 🛠 Requirements
- Cisco Packet Tracer 8.0 or higher
- Basic understanding of networking concepts

---

## 📖 How to Use
1. Clone or download this repository
2. Open any `.pkt` file in Packet Tracer
3. Run simulation mode or real-time mode
4. Explore configurations using CLI

---

## 📈 Learning Path
1. Start with ARP/ICMP (basic connectivity)
2. Move to static routing (router fundamentals)
3. Learn VLANs and segmentation
4. Understand STP/RSTP (loop prevention)
5. Master inter-VLAN routing (Layer 3 switches)
