## 🛠 Requirements

## 📖 How to Use

1. **Clone or download** this repository
2. **Open** any `.pkt` file in Packet Tracer
3. **Run Simulation Mode** to see packet flow and protocol events
4. **Observe** ARP requests, replies, and ICMP echo messages

---

## 🔬 Lab Details: ARP and ICMP Test

### Topology
- **PC0:** 192.168.10.1
- **PC3:** 192.168.10.4
- **Switch0:** 2960-24TT
- **Switch1:** 2960-24TT

### What This Lab Demonstrates
- How **ARP** resolves IP addresses to MAC addresses
- How **ICMP** echo requests (ping) work between hosts
- Switch forwarding and MAC address table learning

### Simulation Events Captured
- ARP requests from source PC
- ARP replies from destination PC
- ICMP echo requests and replies
