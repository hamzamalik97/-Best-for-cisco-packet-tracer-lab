# 🖧 My Cisco Packet Tracer Lab Repository

Welcome to my Cisco networking lab repository!

---

## 📁 Labs Included

| # | File | Description | Protocols | Devices |
|---|------|-------------|-----------|---------|
| 1 | ICMP.ARP.pkt | ARP and ICMP connectivity test | ARP, ICMP | 2x PCs, 2x 2960 switches |
| 2 | static-routing-lab.pkt | Static routing between 3 networks | ICMP, Static Routing | Router0, 4x PCs, 2960 switch |
| 3 | vlan-segmentation-lab.pkt | VLAN segmentation with multiple PCs | VLAN, Ethernet | 50+ PCs, managed switches |
| 4 | stp-bpduguard-lab.pkt | STP, Root Bridge election, BPDU Guard | STP, BPDU Guard, 802.1D | 3x switches |
| 5 | rstp-root-bridge-lab.pkt | RSTP root bridge election, port priority, convergence | RSTP (802.1w), STP | Multiple switches |

---

## 🛠 Requirements
- Cisco Packet Tracer 8.0 or higher

---

## 📖 How to Use
1. Clone or download this repository
2. Open any `.pkt` file in Packet Tracer
3. Run simulation mode
