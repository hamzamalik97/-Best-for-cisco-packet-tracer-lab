# 🖧 My Cisco Packet Tracer Lab Repository

Welcome to my Cisco networking lab repository!

---

## 📁 Labs Included

| File | Description | Protocols | Devices |
|------|-------------|-----------|---------|
| ICMP.ARP.pkt | ARP and ICMP connectivity test | ARP, ICMP | 2x PCs, 2x 2960 switches |
| static-routing-lab.pkt | Static routing between 3 networks | ICMP, Static Routing | Router0, 4x PCs, 2960 switch |
| vlan-segmentation-lab.pkt | VLAN segmentation with multiple PCs | VLAN, Ethernet | 50+ PCs, managed switches |
| stp-bpduguard-lab.pkt | STP, Root Bridge election, and BPDU Guard | STP, BPDU Guard, 802.1D | 3x switches |

---

## 🛠 Requirements
- Cisco Packet Tracer 8.0 or higher

---

## 📖 How to Use
1. Clone or download this repository
2. Open any `.pkt` file in Packet Tracer
3. Run simulation mode
